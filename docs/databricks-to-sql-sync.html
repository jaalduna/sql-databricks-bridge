Event Lifecycle

Operation Details

INS
INSERT

  1. Read all rows from source_table
  2. Validate PK columns exist in data
  3. Check for duplicate primary keys
  4. bulk_insert() into SQL Server
  5. Compare rows_affected vs expected
sync/operations.py : _execute_insert()

UPD
UPDATE

  1. Read all rows from source_table
  2. Validate PK columns exist
  3. For each row, build SET + WHERE clauses
  4. execute_write() per row on SQL Server
  5. Sum total rows_affected
sync/operations.py : _execute_update()

DEL
DELETE

  1. Build WHERE from filter_conditions
  2. COUNT rows matching filter
  3. Validate against max_delete_rows (10k)
  4. execute_write(DELETE) on SQL Server
  5. Compare rows_affected vs counted
sync/operations.py : _execute_delete()
